User Stories

Story 1 

    As a player, I want to have a mana system implemented for caster classes so that it won't be as much as a hassle to use those class abilities.

    Elaboration: I feel it would be a pain to look for the item/items needed to perform the spell effect.

    Constraints: None

    Effort Estimation: 1 person-hours

    Acceptance Test: Having the spell effect tied to a key bind and not an item, without allowing it to be spammed indefinitely.

Story 2 

    As a player, I want to have more armor enchantments and damage types so that there can be a bit more strategy when it comes to dealing with mobs.

    Elaboration: Certain mobs should deal something different from default damage (i.e. spiders should do poison damage and wither skeletons should do necrotic damage). To combat this, different armor enchantments could be added to gain resistance to these damage types. You could add stuff like a "thorns" effect to armor as well.

    Constraints: none

    Effort Estimation: 10 person-hours

    Acceptance Test: Spiders do poison damage and poison resistance will reduce this damage and/or wither skeletons will do necrotic damage and necrotic resistance will reduce this damage.

Story 3 

    As a third-party server owner, I want to be able to use this mod on a multiplayer server.
    
    Elaboration: Classes are independent to-player, each player can go on their own path in the game. Mod works and is executable on multiplayer servers.
    
    Constraints: Independant data storage, or metadata per-player. (Serverside handling is done by forge itself.)
    
    Effort Estimation: 30 minutes -> 2 person-hours.
    
    Acceptance Test: Mod runs on a server-environment, players are individual classes to one-another.

Story 4 

    As a player, I want to be able to play as different classes that give me different abilties.

Story 4.1 

    As a player, I want to have a fighter class.

    Elaboration: The mod should have a class that improves fighting abilities.

    Constraints: None

    Effort Estimation: 5 person-hours

    Acceptance Test: Mod contains a playable fighter class that has buffs and equipment which help fighting ability.

Story 4.2 

    As a player, I want to have a rogue class.

    Elaboration: The mod should have a class that improves sneaking/roguelike abilities.

    Constraints: None

    Effort Estimation: 5 person-hours

    Acceptance Test: Mod contains a playable rogue class that has buffs and equipment which help with sneaking and rougelike activites.

Story 4.3 

    As a player, I want to have a cleric class.

    Elaboration: The mod should have a class that gives healing abilities.

    Constraints: None

    Effort Estimation: 5 person-hours

    Acceptance Test: Mod contains a playable cleric class that has buffs and equipment which allow the player to heal themselves and others.

Story 4.4 

    As a player, I want to have a magic class.

    Elaboration: The mod should have a class that allows the player to use magic.

    Constraints: None

    Effort Estimation: 5 person-hours

    Acceptance Test: Mod contains a playable magic class that has buffs and equipment which allow them to do magic.

Story 5 

    As a player, I want to be able to pick my class with a GUI.
    
    Elaboration: The mod should have a GUI that's accessible either by hotkey or automatically when spawning that allows players to select the class that they want to play as.
    
    Constraints: None
    
    Effort Estimation: 2.5 person-hours
    
    Acceptance Test: Mod contains a bug-free GUI for class selection

Story 6 

    As a player, I want to be able to get more powerful as I play.
    
    Elaboration: The mod should allow players to get more powerful in some way as they play with this mod. This could be with gaining new abilities or more health.
    
    Constraints: None
    
    Effort Estimation: 5 person-hours + 2-5 person-hours per class
    
    Acceptance Test: Players can gain new abilities and features the longer they play with the mod installed.
